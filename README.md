# Getting Started

## Installation
```sh
npm i
```

## Running in Production
```sh
export JWT_ACCESS_TOKEN_SECRET='super-secret-key-1'
export JWT_REFRESH_TOKEN_SECRET='super-secret-key-2'
npm start
```

## Running in Development
```sh
export JWT_ACCESS_TOKEN_SECRET='super-secret-key-1'
export JWT_REFRESH_TOKEN_SECRET='super-secret-key-2'
npm run dev
```

## Running Tests (requires MongoDB)
```sh
export JWT_ACCESS_TOKEN_SECRET='super-secret-key-1'
export JWT_REFRESH_TOKEN_SECRET='super-secret-key-2'
npm run test
```
